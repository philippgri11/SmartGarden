# SmartGarden Version v1.0.0

Kurz gesagt: SmartGarden ist jetzt besser vorbereitet, um neue Versionen verstaendlich per Mail anzukuendigen.

## Was ist neu?

- Bei einer neuen Version kann automatisch eine Changelog-Mail verschickt werden.
- Die Mail nutzt diese Datei als gut lesbare Zusammenfassung.
- Technische Details aus Git bleiben im Hintergrund.

## Was wurde verbessert?

- Wichtige Aenderungen koennen vor dem Veröffentlichen in Alltagssprache beschrieben werden.
- Empfaenger bekommen eine kurze, nachvollziehbare Mail statt einer Liste technischer Commit-Namen.

## Muss ich etwas tun?

- Nein. Wenn eine neue Version markiert wird, verschickt GitHub Actions die Mail automatisch.
- Vor groesseren Aenderungen sollte diese Datei aktualisiert werden.
